# 作业反思：中国乡村美术馆地图

## 项目背景

选择这个题目是因为对乡村美术馆这一文化空间的好奇。中国大量优秀的美术馆散落在乡村中，却没有一个集中的可视化平台进行展示。我希望通过地图的形式，让这些文化场所"被看见"。

## 技术实现

项目使用 HTML/CSS/JavaScript 结合高德地图 API 构建。数据来源于公开渠道收集的 96 家乡村美术馆，通过 Python 脚本调用高德地理编码 API 获取经纬度坐标。

视觉风格上尝试了**像素艺术（Pixel Art）**方向，采用高对比度的荧光配色（绿/粉/黄）和 Silkscreen 像素字体，配合扫描线、粒子爆发等动画效果，试图营造一种"数字考古"的质感——用复古像素美学来呈现当代乡村文化景观。

## 迭代过程

从最初朴素的列表+地图版本（map.html），逐步迭代到具有完整视觉系统的像素风格版本（index.html）。期间最大的调整是侧边栏与地图的比例关系，以及加载动画的节奏感。通过 Claude 的协助，将原本分散的功能整合为单文件应用，便于部署。

## 困难与反思

1. **数据清洗**：原始地址中存在大量重复和格式不统一的条目，手动清理比预期耗时。
2. **API 限制**：高德地图的地理编码对模糊地址的识别率有限，部分场馆只能依赖人工定位。
3. **风格统一**：像素风格的 UI 与现代地图控件的视觉冲突需要大量 CSS 覆盖，在"美观"与"可用性"之间反复权衡。

这次作业让我意识到，vibe coding 的核心不是让 AI 代劳，而是借助快速迭代的能力，把更多精力放在"想做什么"而非"怎么做"上。地图只是一个载体，真正想表达的是：乡村不只有风景，还有被低估的艺术现场。
