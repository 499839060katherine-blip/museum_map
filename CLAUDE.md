# 中国乡村美术馆地图 - 项目文档

## 项目概述

一个展示中国乡村美术馆的交互式地图应用，采用**像素艺术（Pixel Art）**风格设计，使用高德地图 API 进行地理可视化。

## 核心特性

### 视觉风格
- **主题**: Pixelating Convergence（像素聚合）硬核像素风格
- **配色方案**:
  - 荧光绿: `#00ff6a` (--pixel-green)
  - 荧光粉: `#ff99cc` (--pixel-pink)
  - 荧光黄: `#fff34d` (--pixel-yellow)
  - 背景白: `#f0f0f0` (--bg-white)
  - 文本黑: `#000000` (--text-black)
- **字体**: Silkscreen（像素字体）、Noto Sans SC（中文）
- **设计元素**:
  - 硬边框（4px solid black）
  - 像素网格背景（20px）
  - CRT 扫描线效果
  - 噪点纹理叠加

### 页面布局
- **侧边栏宽度**: 480px（占比较大）
- **地图区域**: 自适应剩余空间
- **响应式**: 桌面端优先

### 动画效果

#### 1. 加载动画 - 像素喷泉（Pixel Fountain）
- 中心脉冲扩散动画
- 像素块向四周喷涌
- 计数器显示生成的像素块数量
- 使用 `steps()` 实现抽帧效果

#### 2. 侧边栏动效
- **省份扫描**: 底部扫描线动画，省份名称轮播显示
- **数字乱码**: 统计数字从乱码逐渐显示为正确数值
- **像素进度条**: 10个像素块依次填充，粉色表示高亮
- **卡片扫描**: 列表项从上到下的扫描线进入动画

#### 3. 地图交互
- **标记点**: 16px 荧光绿方块，黑色硬边框
- **点击爆发**: 点击标记时产生8个像素粒子向四周扩散
- **打字机效果**: 信息窗口逐字显示简介（30ms/字符）
- **光标闪烁**: 打字完成后光标持续闪烁

#### 4. 卡片交互
- **悬停效果**: 背景变绿，产生硬阴影位移
- **高亮状态**: 选中卡片为荧光绿背景
- **平滑滚动**: 点击后卡片自动滚动到可视区域

## 注意事项

1. **文件协议**: 本地打开时使用 `file://` 协议，某些浏览器可能有安全限制
2. **Canvas 警告**: 高德地图 API 会产生 Canvas2D 性能警告（可忽略）
3. **API 限制**: 高德 1.4.15 没有 `setMinZoom` 方法，需在事件回调中手动限制
4. **类名操作**: `classList.add()` 不支持带空格的多个类名，需分开调用

## 后续修改建议

### 样式调整
- 侧边栏宽度在 CSS 第 ~70 行 `#sidebar` 处修改
- 颜色变量在 CSS `:root` 中定义
- 动画时长在对应 `@keyframes` 中调整

### 功能扩展
- 添加新的动画效果时，注意使用 `steps()` 保持像素风格
- 修改地图配置在 `initMap()` 函数中
- 数据更新直接替换 `museums_clean.json` 内容

### 常见问题
1. 如果地图不显示，检查 API Key 是否有效
2. 如果搜索无效，检查 `allMuseums` 全局变量是否正确赋值
3. 如果动画卡顿，减少同时运行的动画数量或延长间隔时间
